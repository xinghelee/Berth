FEATURES

终端老手需要的,都在原生里

连接复用与无限分屏

⌘T 新标签、⌘D 分屏共享同一条 SSH 通道,不再重新握手。分屏可无限嵌套,exit 即收起。

跳板机与端口转发

链式跳板一路到底;本地 / 远程 / 动态 SOCKS5 转发,支持 HTTP 与 SOCKS5 代理出站。

密钥与 Touch ID

生成、导入密钥,使用私钥前 Touch ID 验证。known_hosts 指纹确认,变更即警告。

SFTP 文件面板

复用会话连接,拖拽上传下载。远端文件用本地编辑器打开,保存自动回传;chmod、书签、文本预览。

断线不慌

指数退避自动重连,重连后回到原工作目录(OSC 7);命令退出码直接标在终端里(OSC 133)。

生产环境警戒

按主机警戒配色,一眼分清测试与生产。广播输入同时操作多台;Snippets 片段库带 {{变量}}。

SECURITY

三条底线,写进架构

  • 机密只进 Keychain

    密码、passphrase、私钥存 macOS Keychain,任何情况不落盘明文。JSON 备份只含主机结构,不含机密。

  • 数据全在本地

    没有账号,没有云端。主机列表在你的 Mac 上,ssh_config 双向同步,随时整体带走。

  • 签名与公证

    Developer ID 签名 + Apple 公证分发,不上 App Store——沙盒会挡住 ~/.ssh,我们不做阉割版。

FAQ

常见问题

支持哪些系统?
macOS 15 Sequoia 及以上,Apple Silicon 与 Intel 均支持。
为什么不上 App Store?
App Store 沙盒无法读取 ~/.ssh 目录,做不了 ssh_config 同步和密钥导入。Berth 走 Developer ID 签名 + Apple 公证的 DMG 分发,安全性同样有保障。
我的密码存在哪里?
只存 macOS Keychain,由系统加密管理。Berth 没有服务器,也就无从上传。
会有多设备同步吗?
CloudKit 同步在路线图上——依然不经过我们的服务器,数据只在你的 iCloud 里。
